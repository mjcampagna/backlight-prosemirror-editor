Script Tag Usage Examples

Method 1: ES Modules (Recommended)

Use ES6 imports for modern browsers:

<script type="module">
  import { initProseMirrorEditor } from "./dist/prosemirror-bundle.esm.js";
  initProseMirrorEditor(".modern-editor");
</script>

Method 2: Script Tag (No Modules)

Use IIFE bundle for older browsers or when modules aren't available:

<script src="./dist/prosemirror-bundle.iife.js"></script>
<script>
  PMBundle.initProseMirrorEditor(".script-editor");
</script>
